## Meta

The v4..v8 debug session was three days chasing a Dolt journal
theory that never had legs. Root cause was the bench binary
discovery: `filepath.Abs("defn")` + silent PATH fallback + a leftover
binary in the source dir. Fix in `main.go`: rebuild by default, log
the path being used, honor `DEFN_BIN` for explicit override.

Feedback to internalize: when a bug reproduces in one harness but not
another, verify the harnesses are running the same binary before
theorizing about the code under test.
